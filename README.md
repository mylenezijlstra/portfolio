# Modern Portfolio Boilerplate

Dit is een strakke, moderne en premium portfoliowebsite-boilerplate ontworpen en ontwikkeld voor **Mylène Zijlstra**. 

## 🚀 Technologieën & Features

- **Core**: PHP & HTML5 voor flexibiliteit en dynamische datumverwerking.
- **Styling**: Vanilla CSS3 met CSS Custom Properties (variabelen), responsive grids/flexbox, en vloeiende transitie-effecten.
- **Interactie & Visuals**: 
  - Dynamic gradient-blur background blobs die subtiel zweven op de achtergrond.
  - Scroll-reveal animaties die content soepel laten binnenschuiven bij het scrollen.
  - Responsive hamburger-menu voor mobiel/tablet weergaven.
  - Active-link scroll spying om de navigatie-links automatisch te highlighten als je langs secties scrollt.
  - Moderne iconen via [Lucide Icons](https://lucide.dev/).
- **Aesthetic**: Premium dark mode met subtiele noise textures en verfijnde paarse en cyaan kleuraccenten.

## 📂 Projectstructuur

```text
portfolio/
├── assets/
│   ├── css/
│   │   └── style.css      # Core design system, grids en responsieve stijlen
│   ├── js/
│   │   └── main.js        # Scroll-reveal, menu-toggle en scroll-spy functionaliteit
│   └── docs/              # Map voor eventuele CV's of documenten
├── index.php              # Hoofdpagina met alle portfolio-secties
└── README.md              # Projectdocumentatie
```

## 🛠️ Aan de slag

1. Open XAMPP en start de Apache-server.
2. Zorg dat dit project zich bevindt in de `htdocs/portfolio` directory.
3. Open je browser en navigeer naar `http://localhost/portfolio/`.
